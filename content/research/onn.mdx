The **Ontology Neural Network** (ONN) program studies learning systems in
which the internal state is not a flat vector but a typed object carrying
ontological structure. This lets us analyse representations with tools from
algebraic topology and type theory, and — in turn — to reason about the
control laws that close the loop around such a perceiver.

## Current threads

- **Representational cohomology.** How invariants of a learned ontology
  manifest as cohomology classes, and what this implies about
  generalisation.
- **Predicate-binding gates.** Executable operators that compose typed
  features under delay — the practical machinery of the ORTSF framework.
- **Dependence-aware confidence.** Closure theorems for confidence radii
  in a world where features are not independent.

Each of these lives in the journal and, in more polished form, in the
papers section.
