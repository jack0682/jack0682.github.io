## A.1 Standing Assumptions (Axioms A0--A5)

| Label | Axiom | Content |
|-------|-------|---------|
| **A0** | Gauge group | A compact Lie group $G$ with Lie algebra $\mathfrak{g}$, identity $e$, and bi-invariant metric $d_G$ |
| **A1** | Node set | A finite set $V$ with $n := |V|$ |
| **A2** | Time set | A totally ordered set $\mathbb{T}$ |
| **A3** | Fruit threshold | $\theta \in (0,1)$ |
| **A4** | Door threshold | $\tau > 0$ |
| **A5** | Intrinsic data axiom | The degree $d_t(i)$ is included in a fruit's intrinsic data (Axiom 6.1) |

## A.4 Conventions

- Numbering: Definition/Theorem/Lemma X.Y means Chapter X, item Y.
- Gauge action is a **right action**: $(g^{h_1})^{h_2}=g^{h_2 h_1}$.
- $d_G$ always denotes the bi-invariant geodesic distance on $G$.
- $\bar S := V\setminus S$ for any $S\subset V$.
- All sums over edges are over ordered pairs unless stated otherwise.
- "Sketch" proofs explicitly list unverified assumptions.
